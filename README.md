# DSA Sorting Visualizer

An interactive web-based visualization tool developed as a Data Structures & Algorithms (DSA) course project. This application demonstrates how different sorting algorithms work through real-time animations, helping users understand algorithm behavior, comparisons, and element movements.

## Features

* Interactive visualization of sorting algorithms
* Random array generation
* Adjustable animation speed
* Step-by-step execution mode
* Pause and Resume functionality
* Real-time comparison counter
* Real-time move/swap counter
* Keyboard shortcuts for quick control
* Modern and responsive user interface

## Implemented Algorithms

### Insertion Sort

* Time Complexity:

  * Best Case: O(n)
  * Average Case: O(n²)
  * Worst Case: O(n²)

### Selection Sort

* Time Complexity:

  * Best Case: O(n²)
  * Average Case: O(n²)
  * Worst Case: O(n²)

### Bubble Sort

* Time Complexity:

  * Best Case: O(n)
  * Average Case: O(n²)
  * Worst Case: O(n²)

### Shell Sort

* Time Complexity:

  * Depends on gap sequence
  * Average: Approximately O(n log n)

## User Controls

| Action       | Description                     |
| ------------ | ------------------------------- |
| Shuffle      | Generate a new random array     |
| Start        | Start sorting animation         |
| Pause        | Pause current execution         |
| Step         | Execute one operation at a time |
| Speed Slider | Control animation speed         |

## Keyboard Shortcuts

| Key   | Function           |
| ----- | ------------------ |
| Space | Start / Pause      |
| P     | Pause / Resume     |
| S     | Step Execution     |
| R     | Generate New Array |
